1  SUBROUTINE w3fb10(DLAT1, DLON1, DLAT2, DLON2, BEARD, GCDKM)
2 C$$$ SUBPROGRAM DOCUMENTATION BLOCK
3 C . . . .
4 C SUBPROGRAM: W3FB10 LAT/LONG PAIR TO COMPASS BEARING, GCD
5 C PRGMMR: CHASE ORG: NMC421 DATE:88-10-26
6 C
7 C ABSTRACT: GIVEN A PAIR OF POINTS (1) AND (2) GIVEN BY LATITUDE AND
8 C LONGITUDE, W3FB10 COMPUTES THE BEARING AND GREAT CIRCLE DISTANCE
9 C FROM POINT (1) TO POINT (2) ASSUMING A SPHERICAL EARTH. THE
10 C NORTH AND SOUTH POLES ARE SPECIAL CASES. IF LATITUDE OF POINT
11 C (1) IS WITHIN 1E-10 DEGREES OF THE NORTH POLE, BEARING IS THE
12 C NEGATIVE LONGITUDE OF POINT (2) BY CONVENTION. IF LATITUDE OF
13 C POINT (1) IS WITHIN 1E-10 DEGREES OF THE SOUTH POLE, BEARING IS
14 C THE LONGITUDE OF POINT (2) BY CONVENTION. IF POINT (2) IS WITHIN
15 C 1E-6 RADIANS OF THE ANTIPODE OF POINT (1), THE BEARING WILL BE
16 C SET TO ZERO. IF POINT (1) AND POINT (2) ARE WITHIN 1E-10 RADIANS
17 C OF EACH OTHER, BOTH BEARING AND DISTANCE WILL BE SET TO ZERO.
18 C
19 C PROGRAM HISTORY LOG:
20 C 88-08-29 CHASE, P.
21 C 88-09-23 CHASE, P. FIX DUMB SOUTH POLE ERROR
22 C 88-10-05 CHASE, P. FIX BEARING AMBIGUITY
23 C 90-04-12 R.E.JONES CONVERT TO CFT77 FORTRAN
24 C
25 C USAGE: CALL W3FB10(DLAT1, DLON1, DLAT2, DLON2, BEARD, GCDKM)
26 C
27 C INPUT ARGUMENT LIST:
28 C DLAT1 - REAL LATITUDE OF POINT (1) IN DEGREES NORTH.
29 C DLON1 - REAL LONGITUDE OF POINT (1) IN DEGREES EAST.
30 C DLAT2 - REAL LATITUDE OF POINT (2) IN DEGREES NORTH.
31 C DLON2 - REAL LONGITUDE OF POINT (2) IN DEGREES EAST.
32 C
33 C OUTPUT ARGUMENT LIST:
34 C BEARD - REAL BEARING OF POINT (2) FROM POINT (1) IN
35 C COMPASS DEGREES WITH NORTH = 0.0, VALUES FROM
36 C -180.0 TO +180.0 DEGREES.
37 C GCDKM - REAL GREAT CIRCLE DISTANCE FROM POINT (1) TO
38 C POINT (2) IN KILOMETERS.
39 C
40 C SUBPROGRAMS CALLED: NONE
41 C
42 C REMARKS: ACCORDING TO THE NMC HANDBOOK, THE EARTH'S RADIUS IS
43 C 6371.2 KILOMETERS. THIS IS WHAT WE USE, EVEN THOUGH THE VALUE
44 C RECOMMENDED BY THE SMITHSONIAN METEOROLOGICAL HANDBOOK IS
45 C 6371.221 KM. (I WOULDN'T WANT YOU TO THINK THAT I DIDN'T KNOW
46 C WHAT THE CORRECT VALUE WAS.)
47 C METHOD: THE POLES ARE SPECIAL CASES, AND HANDLED SEPARATELY.
48 C OTHERWISE, FROM SPHERICAL TRIGONOMETRY, THE LAW OF COSINES IS USED
49 C TO CALCULATE THE THIRD SIDE OF THE SPHERICAL TRIANGLE HAVING
50 C SIDES FROM THE POLE TO POINTS (1) AND (2) (THE COLATITUDES).
51 C THEN THE LAW OF SINES IS USED TO CALCULATE THE ANGLE AT POINT
52 C (1). A TEST IS APPLIED TO SEE WHETHER THE ARCSINE RESULT MAY BE
53 C BE USED AS SUCH, GIVING AN ACUTE ANGLE AS THE BEARING, OR WHETHER
54 C THE ARCSINE RESULT SHOULD BE SUBTRACTED FROM PI, GIVING AN OBTUSE
55 C ANGLE AS THE BEARING. THIS TEST IS DERIVED BY CONSTRUCTING A
56 C RIGHT SPHERICAL TRIANGLE USING THE POLE, POINT (2), AND THE
57 C MERIDIAN THROUGH POINT(1). THE LATITUDE OF THE RIGHT-ANGLED
58 C VERTEX THEN PROVIDES A TEST--IF LATITUDE (1) IS GREATER THAN THIS
59 C LATITUDE, THE BEARING ANGLE MUST BE OBTUSE, OTHERWISE ACUTE.
60 C IF THE TWO POINTS ARE WITHIN 1E-6 RADIANS OF EACH OTHER
61 C A FLAT EARTH IS ASSUMED, AND THE FOUR-QUADRANT ARCTANGENT
62 C FUNCTION IS USED TO FIND THE BEARING. THE Y-DISPLACEMENT IS
63 C THE DIFFERENCE IN LATITUDE AND THE X-DISPLACEMENT IS THE
64 C DIFFERENCE IN LONGITUDE TIMES COSINE LATITUDE, BOTH IN RADIANS.
65 C DISTANCE IS THEN THE DIAGONAL.
66 C FUNDAMENTAL TRIGONOMETRIC IDENTITIES ARE USED FREELY, SUCH
67 C AS THAT COS(X) = SIN(PI/2 - X), ETC. SEE ALMOST ANY MATHEMATICAL
68 C HANDBOOK, SUCH AS THE C.R.C. STANDARD MATH TABLES UNDER 'RELATIONS
69 C IN ANY SPHERICAL TRIANGLE', OR THE NATIONAL BUREAU OF STANDARDS
70 C 'HANDBOOK OF MATHEMATICAL FUNCTIONS' UNDER SECTION 4.3.149,
71 C FORMULAS FOR SOLUTION OF SPHERICAL TRIANGLES.
72 C DOUBLE PRECISION IS USED INTERNALLY BECAUSE OF THE WIDE
73 C RANGE OF GEOGRAPHIC VALUES THAT MAY BE USED.
74 C
75 C ATTRIBUTES:
76 C LANGUAGE: CRAY CFT77 FORTRAN
77 C MACHINE: CRAY Y-MP8/832
78 C
79 C$$$
80 C
81 C *** IMPLICIT TYPE DEFAULTS.....
82 C
83  IMPLICIT REAL (A-H,O-Z)
84 C
85 C *** CONSTANTS......
86 C
87  REAL PI
88  REAL HALFPI
89  REAL DR
90  REAL RD
91  REAL TDEG, TRAD, TPOD, TFLT
92  REAL EARTHR
93  REAL WHOLCD, HALFCD, QUARCD
94 C
95 C *** VARIABLES......
96 C
97  REAL RLAT1, RLAT2, COSLA1, COSLA2, SINLA1, SINLA2
98  REAL DLOND, RLOND, COSLO, SINLO, SANGG, ABEAR
99  REAL YDISP, XDISP, DDLAT1, DDLAT2, DBANG
100  REAL DLAT1, DLAT2, DLON1, DLON2, BEARD, GCDKM
101 C
102 C *** CONVERT LATITUDES AND LONGITUDE DIFFERENCE TO RADIANS.
103 C
104  DATA pi /3.141592653589793238462643/
105  DATA halfpi/1.570796326794896619231322/
106  DATA dr /0.017453292519943295769237/
107  DATA rd /57.295779513082320876798155/
108  DATA tdeg /1e-10/, trad/1e-10/, tpod/1e-6/, tflt/1e-6/
109  DATA earthr/6371.2/
110  DATA wholcd/360.0/, halfcd/180.0/, quarcd/90.0/
111 
112  ddlat1 = dlat1
113  ddlat2 = dlat2
114  rlat1 = dr * ddlat1
115  rlat2 = dr * ddlat2
116  dlond = dlon2 - dlon1
117  IF (dlond .GT. halfcd) dlond = dlond - wholcd
118  IF (dlond .LT. -halfcd) dlond = dlond + wholcd
119  rlond = dr * dlond
120 C
121 C *** FIRST WE ATTACK THE CASES WHERE POINT 1 IS VERY CLOSE TO THE
122 C *** NORTH OR SOUTH POLES.
123 C *** HERE WE USE CONVENTIONAL VALUE FOR BEARING.. - LONG (2) AT THE
124 C *** NORTH POLE, AND + LONG (2) AT THE SOUTH POLE.
125 C
126  IF (abs(ddlat1-quarcd) .LT. tdeg) THEN
127  IF (abs(ddlat2-quarcd) .LT. tdeg) THEN
128  dbang = 0.0
129  sangg = 0.0
130  ELSE IF (abs(ddlat2+quarcd) .LT. tdeg) THEN
131  dbang = 0.0
132  sangg = pi
133  ELSE
134  dbang = -dlon2
135  sangg = halfpi - rlat2
136  ENDIF
137  ELSE IF (abs(ddlat1+quarcd) .LT. tdeg) THEN
138  IF (abs(ddlat2-quarcd) .LT. tdeg) THEN
139  dbang = 0.0
140  sangg = pi
141  ELSE IF (abs(ddlat2+quarcd) .LT. tdeg) THEN
142  dbang = 0.0
143  sangg = 0.0
144  ELSE
145  dbang = +dlon2
146  sangg = halfpi + rlat2
147  ENDIF
148 C
149 C *** NEXT WE ATTACK THE CASES WHERE POINT 2 IS VERY CLOSE TO THE
150 C *** NORTH OR SOUTH POLES.
151 C *** HERE BEARING IS SIMPLY 0 OR 180 DEGREES.
152 C
153  ELSE IF (abs(ddlat2-quarcd) .LT. tdeg) THEN
154  dbang = 0.0
155  sangg = halfpi - rlat1
156  ELSE IF (abs(ddlat2+quarcd) .LT. tdeg) THEN
157  dbang = halfcd
158  sangg = halfpi + rlat1
159 C
160 C *** THE CASE REMAINS THAT NEITHER POINT IS AT EITHER POLE.
161 C *** FIND COSINE AND SINE OF LATITUDES AND LONGITUDE DIFFERENCE
162 C *** SINCE THEY ARE USED IN MORE THAN ONE FORMULA.
163 C
164  ELSE
165  cosla1 = cos(rlat1)
166  sinla1 = sin(rlat1)
167  cosla2 = cos(rlat2)
168  sinla2 = sin(rlat2)
169  coslo = cos(rlond)
170  sinlo = sin(rlond)
171 C
172 C *** FOLLOWING IS FORMULA FOR GREAT CIRCLE SUBTENDED ANGLE BETWEEN
173 C *** POINTS IN RADIAN MEASURE.
174 C
175  sangg = acos(sinla1*sinla2 + cosla1*cosla2*coslo)
176 C
177 C *** IF THE GREAT CIRCLE SUBTENDED ANGLE IS VERY SMALL, FORCE BOTH
178 C *** BEARING AND DISTANCE TO BE ZERO.
179 C
180  IF (abs(sangg) .LT. trad) THEN
181  dbang = 0.0
182  sangg = 0.0
183 C
184 C *** IF THE GREAT CIRCLE SUBTENDED ANGLE IS JUST SMALL, ASSUME A
185 C *** FLAT EARTH AND CALCULATE Y- AND X-DISPLACEMENTS. THEN FIND
186 C *** BEARING USING THE ARCTANGENT FUNCTION AND DISTANCE USING THE
187 C *** SQUARE ROOT.
188 C
189  ELSE IF (abs(sangg) .LT. tflt) THEN
190  ydisp = rlat2-rlat1
191  xdisp = rlond*cosla2
192  abear = atan2(xdisp, ydisp)
193  dbang = rd*abear
194  sangg = sqrt(ydisp**2 + xdisp**2)
195 C
196 C *** IF THE ANGLE IS RATHER CLOSE TO PI RADIANS, FORCE BEARING TO
197 C *** BE ZERO AND DISTANCE TO BE PI.
198 C *** THE TEST FOR 'CLOSE TO PI' IS MORE RELAXED THAN THE TEST FOR
199 C *** 'CLOSE TO ZERO' TO ALLOW FOR GREATER RELATIVE ERROR.
200 C
201  ELSE IF (abs(sangg-pi) .LT. tpod) THEN
202  dbang = 0.0
203  sangg = pi
204 C
205 C *** OTHERWISE COMPUTE THE PRINCIPAL VALUE OF THE BEARING ANGLE
206 C *** USING THE LAW OF SINES. THE DIVISION BY THE SINE FORCES US TO
207 C *** LIMIT THE DOMAIN OF THE ARCSINE TO (-1,1).
208 C
209  ELSE
210  abear = asin(amax1(-1.0,amin1(+1.0,cosla2*sinlo/
211  & sin(sangg))))
212 C
213 C *** IF THE LONGITUDE DIFFERENCE IS LESS THAN PI/2 IT IS NECESSARY
214 C *** TO CHECK WHETHER THE BEARING ANGLE IS ACUTE OR OBTUSE BY
215 C *** COMPARING LATITUDE (1) WITH THE LATITUDE OF THE GREAT CIRCLE
216 C *** THROUGH POINT (2) NORMAL TO MERIDIAN OF LONGITUDE (1). IF
217 C *** LATITUDE (1) IS GREATER, BEARING IS OBTUSE AND THE ACTUAL
218 C *** BEARING ANGLE IS THE SUPPLEMENT OF THE ANGLE CALCULATED ABOVE.
219 C
220  IF (0.0 .LE. cosla1*sinla2 .AND. cosla1*sinla2 .LE.
221  & cosla2*sinla1*coslo .OR. cosla1*sinla2 .LE. 0.0 .AND.
222  & cosla2*sinla1*coslo .GE. cosla1*sinla2) abear =
223  & sign(pi,abear) - abear
224  dbang = rd * abear
225  ENDIF
226  ENDIF
227 C
228 C *** THIS FINISHES THE CASE WHERE POINTS ARE NOT AT THE POLES.
229 C *** NOW CONVERT BEARING TO DEGREES IN RANGE -180 TO +180 AND FIND
230 C *** GREAT CIRCLE DISTANCE IN KILOMETERS.
231 C
232  IF (dbang .LE. -halfcd) dbang = dbang + wholcd
233  IF (dbang .GT. halfcd) dbang = dbang - wholcd
234  gcdkm = earthr * sangg
235  beard = dbang
236  RETURN
237  END
