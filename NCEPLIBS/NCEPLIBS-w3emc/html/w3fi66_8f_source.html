4 C SUBPROGRAM: W3FI66 OFFICE NOTE 29 REPORT BLOCKER
5 C PRGMMR: KEYSER ORG: NMC22 DATE:92-06-29
6 C
7 C ABSTRACT: BLOCKS REPORTS WHICH HAVE BEEN PACKED INTO NMC OFFICE
8 C NOTE 29 CHARACTER FORMAT INTO FIXED-LENGTH RECORDS. A REPORT
9 C CANNOT SPAN TWO RECORDS; IF THERE IS NOT ENOUGH ROOM TO FIT
10 C THE CURRENT REPORT IN THE RECORD, THE SUBROUTINE RETURNS TO
11 C THE CALLING PROGRAM WITHOUT ANY MOVEMENT OF DATA.
12 C
13 C PROGRAM HISTORY LOG:
14 C 90-01-?? L. MARX, UNIV. OF MD -- CONVERTED CODE FROM ASSEMBLER
15 C TO VS FORTRAN; EXPANDED ERROR RETURN CODES IN 'NFLAG'
16 C 91-08-23 D. A. KEYSER, NMC22 -- USE SAME ARGUMENTS AS W3AI05;
17 C STREAMLINED CODE; DOCBLOCKED AND COMMENTED; DIAG-
18 C NOSTIC PRINT FOR ERRORS
19 C 92-06-29 D. A. KEYSER W/NMC22 -- CONVERT TO CRAY CFT77 FORTRAN
20 C
21 C USAGE: CALL W3FI66(COCBUF,COCBLK,NFLAG,NSIZE)
22 C INPUT ARGUMENT LIST:
23 C COCBUF - CHARACTER*10 ARRAY CONTAINING A SINGLE PACKED REPORT
24 C - IN OFFICE NOTE 29/124 FORMAT.
25 C COCBLK - CHARACTER*10 ARRAY HOLDING A BLOCK OF PACKED REPORTS
26 C - UP TO AND INCLUDING THE PREVIOUS ONE
27 C NFLAG - MARKER INDICATING RELATIVE LOCATION (IN BYTES)
28 C - OF END OF LAST REPORT IN COCBLK. EXCEPTION:
29 C - NFLAG MUST BE SET TO ZERO PRIOR TO BLOCKING THE FIRST
30 C - PACKED REPORT INTO A NEW BLOCK. SUBSEQUENTLY, THE
31 C - VALUE OF NFLAG RETURNED BY THE PREVIOUS CALL TO W3FI66
32 C - SHOULD BE USED AS INPUT. (SEE OUTPUT ARGUMENT LIST
33 C - BELOW.) IF NFLAG IS NEGATIVE, W3FI66 WILL RETURN
34 C - IMMEDIATELY WITHOUT ACTION.
35 C NSIZE - MAXIMUM NUMBER OF CHARACTERS IN COCBLK ARRAY
36 C (SHOULD BE A MULTIPLE OF 4)
37 C
38 C OUTPUT ARGUMENT LIST:
39 C COCBLK - CHARACTER*10 ARRAY HOLDING A BLOCK OF PACKED REPORTS
40 C - UP TO AND INCLUDING THE CURRENT ONE
41 C NFLAG - MARKER INDICATING RELATIVE LOCATION (IN BYTES)
42 C - OF END OF CURRENT REPORT IN COCBLK. NFLAG
43 C - WILL BE SET TO -1 IF W3FI66 CANNOT FIT THE CURRENT
44 C - PACKED REPORT INTO THE REMAINDER OF THE BLOCK (I.E.,
45 C - THE BLOCK IS FULL). NFLAG WILL NOT CHANGE FROM ITS
46 C - INPUT ARGUMENT VALUE IF THE STRING "END REPORT" IS
47 C - NOT FOUND AT THE END OF THE CURRENT REPORT. (CURRENT
48 C - PACKED REPORT HAS INVALID LENGTH AND IS NOT BLOCKED)
49 C
50 C OUTPUT FILES:
51 C FT06F001 - PRINTOUT
52 C
53 C REMARKS: THE USER MUST SET NFLAG TO ZERO EACH TIME THE ARRAY IS
54 C TO BE FILLED WITH PACKED REPORTS IN OFFICE NOTE 29/124 FORMAT.
55 C W3FI66 WILL THEN INSERT THE FIRST REPORT AND FILL THE REMAINDER
56 C OF THE OUTPUT ARRAY COCBLK WITH THE STRING 'END RECORD'.
57 C AN ATTEMPT IS MADE TO INSERT A REPORT IN THE OUTPUT ARRAY
58 C EACH TIME W3FI66 IS CALLED. IF THE REMAINING PORTION OF THE
59 C OUTPUT ARRAY IS NOT LARGE ENOUGH TO HOLD THE CURRENT REPORT,
60 C W3FI66 SETS NFLAG TO -1. THE USER SHOULD THEN OUTPUT THE
61 C BLOCKED RECORD, SET NFLAG TO ZERO, AND CALL W3FI66 AGAIN WITH
62 C THE SAME REPORT IN THE INPUT ARRAY.
63 C AFTER A GIVEN REPORT IS SUCCESSFULLY BLOCKED INTO COCBLK,
64 C W3FI66 SETS NFLAG AS A POINTER FOR THE NEXT REPORT TO BE BLOCKED.
65 C THIS POINTER IS A RELATIVE ADDRESS AND A CHARACTER COUNT.
66 C THE THREE CHARACTERS SPECIFYING THE LENGTH OF THE REPORT
67 C ARE CHECKED FOR VALID CHARACTER NUMBERS AND THE VALUE IS TESTED
68 C FOR POINTING TO THE END OF THE REPORT (STRING "END REPORT"). IF
69 C INVALID, THE REPORT IS NOT INSERTED INTO THE BLOCK AND THERE IS
70 C AN IMMEDIATE RETURN TO THE USER. IN THIS CASE, THE VALUE OF
71 C NFLAG DOES NOT CHANGE FROM ITS INPUT VALUE.
72 C

74 C ASSEMBLY LANGUAGE CODE IN CRAY W3LIB.
