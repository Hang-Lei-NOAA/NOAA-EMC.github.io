1  SUBROUTINE w3fb05(XI,XJ,XMESHL,ORIENT,ALAT,ALONG)
2 C$$$ SUBPROGRAM DOCUMENTATION BLOCK
3 C
4 C SUBPROGRAM: W3FB05 GRID COORDINATES TO LATITUDE, LONGITUDE
5 C AUTHOR: JONES,R.E. ORG: W345 DATE: 86-07-17
6 C
7 C ABSTRACT: CONVERTS THE COORDINATES OF A LOCATION FROM THE GRID(I,J)
8 C COORDINATE SYSTEM OVERLAID ON THE POLAR STEREOGRAPHIC MAP PROJEC-
9 C TION TRUE AT 60 DEGREES N OR S LATITUDE TO THE NATURAL COORDINATE
10 C SYSTEM OF LATITUDE/LONGITUDE ON THE EARTH. W3FB05 IS THE REVERSE
11 C OF W3FB04.
12 C
13 C PROGRAM HISTORY LOG:
14 C 86-07-17 R.E.JONES
15 C 89-11-01 R.E.JONES CHANGE TO CRAY CFT77 FORTRAN
16 C
17 C USAGE: CALL W3FB05 (XI, XJ, XMESHL, ORIENT, ALAT, ALONG)
18 C
19 C INPUT VARIABLES:
20 C NAMES INTERFACE DESCRIPTION OF VARIABLES AND TYPES
21 C ------ --------- -----------------------------------------------
22 C XI ARG LIST I OF THE POINT RELATIVE TO THE NORTH OR S. POLE
23 C XJ ARG LIST J OF THE POINT RELATIVE TO THE NORTH OR S. POLE
24 C XMESHL ARG LIST MESH LENGTH OF GRID IN KM AT 60 DEGREES(<0 IF SH)
25 C (190.5 LFM GRID, 381.0 NH PE GRID,-381.0 SH PE GRID)
26 C ORIENT ARG LIST ORIENTATION WEST LONGITUDE OF THE GRID
27 C (105.0 LFM GRID, 80.0 NH PE GRID, 260.0 SH PE GRID)
28 C
29 C OUTPUT VARIABLES:
30 C NAMES INTERFACE DESCRIPTION OF VARIABLES AND TYPES
31 C ------ --------- -----------------------------------------------
32 C ALAT ARG LIST LATITUDE IN DEGREES (<0 IF SH)
33 C ALONG ARG LIST WEST LONGITUDE IN DEGREES
34 C
35 C SUBPROGRAMS CALLED:
36 C NAMES LIBRARY
37 C ------------------------------------------------------- --------
38 C ASIN ATAN2 SYSLIB
39 C
40 C REMARKS: ALL PARAMETERS IN THE CALLING STATEMENT MUST BE
41 C REAL. THE RANGE OF ALLOWABLE LATITUDES IS FROM A POLE TO
42 C 30 DEGREES INTO THE OPPOSITE HEMISPHERE.
43 C THE GRID USED IN THIS SUBROUTINE HAS ITS ORIGIN (I=0,J=0)
44 C AT THE POLE, SO IF THE USER'S GRID HAS ITS ORIGIN AT A POINT
45 C OTHER THAN A POLE, A TRANSLATION IS REQUIRED TO GET I AND J FOR
46 C INPUT INTO W3FB05. THE SUBROUTINE GRID IS ORIENTED SO THAT
47 C GRIDLINES OF I=CONSTANT ARE PARALLEL TO A WEST LONGITUDE SUP-
48 C PLIED BY THE USER. THE EARTH'S RADIUS IS TAKEN TO BE 6371.2 KM.
49 C
50 C WARNING: THIS CODE WILL NOT VECTORIZE, IT IS NORMALY USED IN A
51 C DOUBLE DO LOOP WITH W3FT01, W3FT00, ETC. TO VECTORIZE IT,
52 C PUT IT IN LINE, PUT W3FT01, W3FT00, ETC. IN LINE.
53 C
54 C LANGUAGE: CRAY CFT77 FORTRAN
55 C MACHINE: CRAY Y-MP8/832
56 C
57 C$$$
58 C
59  DATA degprd/57.2957795/
60  DATA earthr/6371.2/
61 C
62  gi2 = ((1.86603 * earthr) / (xmeshl))**2
63  r2 = xi * xi + xj * xj
64 C
65  IF (r2.EQ.0.0) THEN
66  along = 0.0
67  alat = 90.0
68  IF (xmeshl.LT.0.0) alat = -alat
69  RETURN
70  ELSE
71  alat = asin((gi2 - r2) / (gi2 + r2)) * degprd
72  angle = degprd * atan2(xj,xi)
73  IF (angle.LT.0.0) angle = angle + 360.0
74  ENDIF
75 C
76  IF (xmeshl.GE.0.0) THEN
77  along = 270.0 + orient - angle
78 C
79  ELSE
80 C
81  along = angle + orient - 270.0
82  alat = -(alat)
83  ENDIF
84 C
85  IF (along.LT.0.0) along = along + 360.0
86  IF (along.GE.360.0) along = along - 360.0
87 C
88  RETURN
89 C
90  END
