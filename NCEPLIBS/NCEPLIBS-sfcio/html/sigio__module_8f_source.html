5 ! Module: sigio_module API for global spectral sigma file I/O
6 ! Prgmmr: iredell Org: w/nx23 date: 1999-01-18
7 !
8 ! Abstract: This module provides an Application Program Interface
9 ! for performing I/O on the sigma restart file of the global spectral model.
10 ! Functions include opening, reading, writing, and closing as well as
11 ! allocating and deallocating data buffers used in the transfers.
12 ! The I/O performed here is sequential.
13 ! The transfers are limited to header records or data records.

300 ! Remarks:
301 ! (1) The sigma file format follows:
302 ! For ivs=198410:
303 ! ON85 label (32 bytes)
304 ! Header information record containing
305 ! real forecast hour, initial date, sigma interfaces, sigma levels,
306 ! padding to allow for 100 levels, and finally 44 identifier words
307 ! containing JCAP, LEVS, NTRAC, IREALF, etc. (250 4-byte words)
308 ! (word size in the remaining records depends on the value of IREALF)
309 ! Orography (NC words, where NC=(JCAP+1)*(JCAP+2))
310 ! Log surface pressure (NC words)
311 ! Temperature (LEVS records of NC words)
312 ! Divergence & Vorticity interleaved (2*LEVS records of NC words)
313 ! Tracers (LEVS*NTRAC records of NC words)
314 ! Extra grid fields (NXGR records of LONB*LATB words)
315 ! For ivs=200509:
316 ! Label containing
317 ! 'GFS ','SIG ',ivs,nhead,ndata,reserved(3) (8 4-byte words)
318 ! Header records
319 ! lhead(nhead),ldata(ndata) (nhead+ndata 4-byte words)
320 ! fhour, idate(4), jcap, levs, itrun, iorder, irealf, igen,
321 ! latf, lonf, latb, lonb, latr, lonr, ntrac, nvcoord,
322 ! icen2, iens(2), idpp, idsl, idvc, idvm, idvt, idrun, idusr,
323 ! pdryini, ncldt, ixgr, ixga,reserved(17) (50 4-byte words)
324 ! vcoord((levs+1)*nvcoord 4-byte words)
325 ! cfvars(5+ntrac 8-byte character words)
326 ! Data records (word size depends on irealf)
327 ! orography (nc words, where nc=(jcap+1)*(jcap+2))
328 ! log surface pressure (nc words)
329 ! temperature (levs records of nc words)
330 ! divergence (levs records of nc words)
331 ! vorticity (levs records of nc words)
332 ! tracers (levs*ntrac records of nc words)
333 ! scalars (nxss words)
334 ! extra physics grid fields (nxgr records of lonb*latb words)
335 ! extra scalars (nxss words)
336 ! extra dynamics grid fields (nxga records of lonf*latf words)
337 !
338 ! (2) Possible return codes:
339 ! 0 Successful call
340 ! -1 Open or close I/O error
341 ! -2 Header record I/O error (possible EOF)
342 ! -3 Allocation or deallocation error
343 ! -4 Data record I/O error
344 ! -5 Insufficient data dimensions allocated
345 ! -6 Attempted to read a NEMSIO file
346 !
