9 C
10 C ABSTRACT: THIS SUBPROGRAM PERFORMS A SPHERICAL TRANSFORM
11 C FROM SPECTRAL COEFFICIENTS OF SCALAR FIELDS

17 C THE TRANSFORMS ARE ALL MULTIPROCESSED OVER STATIONS.
18 C TRANSFORM SEVERAL FIELDS AT A TIME TO IMPROVE VECTORIZATION.
19 C SUBPROGRAM CAN BE CALLED FROM A MULTIPROCESSING ENVIRONMENT.
