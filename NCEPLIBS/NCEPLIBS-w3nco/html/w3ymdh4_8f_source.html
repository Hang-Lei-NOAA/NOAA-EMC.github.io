4 C SUBPROGRAM: W3YMDH4 4-BYTE DATE WORD UNPACKER AND PACKER
5 C AUTHOR: Brill,K.F. ORG: NP/22 DATE: 98-07-29
6 C
7 C ABSTRACT: OBTAINS THE COMPONENTS OF THE NMC DATE WORD (NCEP Y2K
8 C COMPLIANT FORM), OR GIVEN ITS COMPONENTS, FORMS AN NMC TYPE DATE
9 C WORD. THE PACKING IS DONE USING BASE 32.
10 C
11 C If the first byte of IDATE is less than 101, then the old
12 C Office Note 84 packing is assumed. A four-digit year is
13 C always returned. To pack the "old" way, pass in a 2-digit
14 C year.
15 C
16 C This program will work for the years ranging from A.D. 101
17 C through 79359.
18 C
19 C On unpacking, years less than or equal to 100 are returned
20 C as follows:
21 C
22 C 0-50 2000--2050
23 C 51-100 1951--2000
24 C
25 C
