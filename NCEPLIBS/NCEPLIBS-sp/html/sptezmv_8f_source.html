9 C ABSTRACT: THIS SUBPROGRAM PERFORMS SPHERICAL TRANSFORMS

16 C GRID FIELDS ARE INDEXED EAST TO WEST, THEN NORTH TO SOUTH.
17 C FOR MORE FLEXIBILITY AND EFFICIENCY, CALL SPTRAN.

21 C 96-02-29 IREDELL
