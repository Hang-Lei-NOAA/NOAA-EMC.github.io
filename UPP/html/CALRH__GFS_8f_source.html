114 ! Subprogram: fpvsnew Compute saturation vapor pressure
115 ! Author: N Phillips w/NMC2X2 Date: 30 dec 82
116 !
117 ! Abstract: Compute saturation vapor pressure from the temperature.
118 ! A linear interpolation is done between values in a lookup table
119 ! computed in gpvs. See documentation for fpvsx for details.
120 ! Input values outside table range are reset to table extrema.
121 ! The interpolation accuracy is almost 6 decimal places.
122 ! On the Cray, fpvs is about 4 times faster than exact calculation.
123 ! This function should be expanded inline in the calling routine.
124 !
125 ! Program History Log:
126 ! 91-05-07 Iredell made into inlinable function
127 ! 94-12-30 Iredell expand table
128 ! 1999-03-01 Iredell f90 module
129 ! 2001-02-26 Iredell ice phase
130 !
131 ! Usage: pvs=fpvsnew(t)
132 !
133 ! Input argument list:
134 ! t Real(krealfp) temperature in Kelvin
135 !
136 ! Output argument list:
137 ! fpvsnew Real(krealfp) saturation vapor pressure in Pascals
138 !
139 ! Attributes:
140 ! Language: Fortran 90.
141 !
142 !$$$

144  integer,parameter:: nxpvs=7501

147  real,parameter:: con_cvap =1.8460e+3 ! spec heat H2O gas (J/kg/K)
