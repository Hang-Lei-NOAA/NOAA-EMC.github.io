MOM5/CICE 0.5 degree grid

Description

This page describes a MOM5 grid that is a tripolar at 0.5 degree resolution. The grid is a tripole grid built around half degree spacing. The latitudinal grid at the equator is 0.25 degrees and the tripole begins at about 65N.
